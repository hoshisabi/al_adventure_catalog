# AL Adventure Catalog

A sortable, filterable catalog of D&D Adventurers League adventures published on
[DM's Guild](https://www.dmsguild.com/).

**[Browse the catalog →](https://hoshisabi.com/al_adventure_catalog/)**

## What's here

- **Filter & sort** adventures by season, campaign setting (Forgotten Realms, Eberron,
  Ravenloft, Dragonlance, and more), tier, level range, length, and more.
- **Stats dashboard** — see a breakdown of the catalog by season, campaign, and tier.
- **Private Inventory** — link your own purchased PDFs (e.g. from Google Drive) so the
  catalog shows which adventures you already own. See the
  [Private Inventory Guide](https://hoshisabi.com/al_adventure_catalog/private-inventory-guide/)
  to set it up.

## About the data

Adventure details (titles, codes, authors, hours, tier, etc.) are gathered from publicly
listed DM's Guild product pages. To respect DM's Guild's content rights, only structured
metadata is published here — page text and images aren't republished.
